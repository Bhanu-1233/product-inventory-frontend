# Frontend – Skillwise Inventory UI

## Tech

- React (Vite)
- Axios for API calls
- Custom CSS for a clean, responsive UI

## Setup

```bash
cd frontend
cp .env.example .env
npm install
npm run dev
```

The app runs on `http://localhost:5173` by default.

## Environment variables

- `VITE_API_BASE_URL` – base URL of your backend API  
  Example (local): `http://localhost:4000`  
  Example (production): `https://your-backend.onrender.com`
```
